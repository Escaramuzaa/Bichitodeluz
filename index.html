▶ Instrucciones de instalación (2 minutos)
  1. Descarga el archivo apps_script.js que viene junto a esta página
  2. Ve a script.google.com → "Nuevo proyecto"
  3. Borra todo el código → pega el contenido de apps_script.js
  4. Guarda (Ctrl+S) → ponle nombre "Cafeteria Backend"
  5. Clic en "Implementar" → "Nueva implementación"
  6. Tipo: Aplicación web · Ejecutar como: Yo · Acceso: Cualquier usuario
  7. Clic "Implementar" → autoriza los permisos → copia la URL
  8. Pega la URL abajo y haz clic en "Conectar"
